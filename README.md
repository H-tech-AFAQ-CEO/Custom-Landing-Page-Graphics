# Custom Landing Page & Graphics

A premium, modern landing page built with cutting-edge web technologies. This project showcases a sophisticated design with smooth animations and professional aesthetics.

## Developer & Designer

**Afaq Ahmad** — Developer and Designer

## Technologies Used

- **HTML5** — Semantic markup
- **CSS3** — Custom styling with CSS variables
- **GSAP** — Premium scroll animations
- **ScrollTrigger** — Scroll-based animations
- **Splitting.js** — Text animations
- **CountUp.js** — Animated counters
- **Google Fonts** — Syne & DM Sans typography

## Features

- Modern dark theme design
- Smooth scroll animations
- Interactive elements
- Responsive layout
- Professional typography
- Premium visual effects

## Getting Started

Simply open `index.html` in your browser to view the landing page.

```bash
# Or use a local server
npx serve .
```
